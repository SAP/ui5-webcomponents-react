# AnalyticalTable Plugin: useAnnounceEmptyCells

<ImportStatement moduleNames={['AnalyticalTableHooks']} packageName={'@ui5/webcomponents-react'} />

The `useAnnounceEmptyCells` plugin hook adds screen reader announcements for empty cells.

**Note:** Some screen readers (depending on their configuration) automatically detect empty cells, potentially resulting in duplicate announcements of empty cells.

## Example

<Canvas sourceState="none" of={ComponentStories.PluginAnnounceEmptyCells} />

```jsx
<AnalyticalTable
  data={data}
  columns={columns}
  visibleRows={5}
  tableHooks={[AnalyticalTableHooks.useAnnounceEmptyCells]}
/>
```

**First data entry with empty values or undefined fields:**

```js
{
  age: undefined,
  friend: {
    name: '',
    age: 0
  },
  status: 'Error'
}
```
