## Fully customizable title

<MessageStrip
  design={MessageStripDesign.Warning}
  hideCloseButton
  children={
    <>
      As the <code>title</code> slot name collides with the standard HTML <code>title</code> attribute, using the{' '}
      <code>title</code> prop will not work. <br />
      You must pass the <code>title</code> slot as part of the components <code>children</code>.
    </>
  }
/>

In some cases you might need to fully customize the title element of the `IllustratedMessage`.
You can achieve this by using the `title` slot.

**Note:** Using this slot, the default title text of illustration and the value of `titleText` property will be overwritten.

```jsx
<IllustratedMessage>
  <h5 slot="title">This is my fully customized title</h5>
</IllustratedMessage>
```
