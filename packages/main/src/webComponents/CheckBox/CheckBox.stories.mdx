## Usage

You can manually set the width of the element containing the box and the label using the `width` property. If the text exceeds the available width, it is truncated. The touchable area for toggling the `CheckBox` ends where the text ends.

You can disable the `CheckBox` by setting the `disabled` property to `true`, or use the `CheckBox` in read-only mode by setting the `readonly` property to `true`.
